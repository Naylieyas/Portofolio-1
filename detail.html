§ Project · 01

Kompas Overload.

Dashboard agregasi berita realtime yang menyaring headline berdasar minat pembaca — dibalut dark-mode Neo-Brutalism.

KO
Client

Personal Project

Year

2025

Role

Design & Build

Duration

3 weeks

Tujuan

Menyediakan interface baca berita yang cepat, ringan, dan berkarakter — melawan kejenuhan tampilan portal berita mainstream. Fokus utama: kejelasan hierarki dan responsivitas.

Fitur Utama

  • Filter kategori berbasis pill-button dengan animasi kinetik.
  • Dark mode toggle dengan localStorage persistence.
  • Pencarian instan tanpa reload menggunakan array filter murni.
  • Layout adaptif dari 320px hingga ultrawide.

Teknologi

HTML5 CSS Grid Vanilla JS Fetch API

Tantangan

  • Menyeimbangkan tipografi besar Neo-Brutalism dengan keterbacaan panjang teks berita.
  • Menjaga performa render saat daftar berita melebihi 300 item.
  • Mendesain state kosong (empty state) yang tetap terasa hidup.
